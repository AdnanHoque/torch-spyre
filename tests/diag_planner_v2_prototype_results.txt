# Offline planner v2 prototype

divisibility / stick alignment / EAR ceiling, ranks via V4 cost
model, compares choice to the current planner's pure-M default.

Caveats:
  - Predictions use V4 cost model. Validation residuals are 16% mean
    on the 30-row Project B set; some rows have higher error,
    especially small-M HMI BW and +id K-dependent residuals.
  - Production planner has constraints (memory layout, output
    shapes, etc.) the prototype doesn't enforce — actual planner
    integration may reject some v2 picks.
  - v2 picks should be hardware-verified before any production
    rollout.
