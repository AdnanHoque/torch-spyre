# Core-emission reorder probe — K-split PSUM + M↔N output

# warmup=3 iters=15, fp16, SENCORES=32

## Hypothesis recap

- For (m, 1, k>1): default M-fast scatters K-chain across the
  ring (cores {0, m, 2m, ...}); reverse K-fast packs K-chain
  contiguously (cores {0..k-1}). Predicted hop reduction
  scales with `m*n - 1` per chain step → reverse should win
  iff PSUM is on the critical path.
- For (m, n, 1): no PSUM chain. Default vs reverse swaps which
  output dim is the fast-changing core-ID dim. Earlier probes
  showed null result; we re-confirm on focused shapes.
